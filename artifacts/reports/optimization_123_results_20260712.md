# 75 MHz nanoGPT 优化 1/2/3 实测结果

## 1. FFN 跨组预取

- 计算时钟：75 MHz
- FFN 仿真周期：641,088 -> 493,872
- 45 字符：25.481 s，逐字符一致
- 200 字符：99.080 s，逐字符一致
- 时序：WNS +0.241 ns，TNS 0

## 2. 75/100 MHz 双时钟 DDR

- 计算/控制域：75 MHz
- HP0/DDR AXI 域：100 MHz
- 三个 AXI Clock Converter：核心 DDR、DMA MM2S、DMA S2MM
- 时序：WNS +1.345 ns，TNS 0，WHS +0.032 ns
- 200 字符：113.707 s，逐字符一致
- 结论：异步 FIFO 固定延迟使短 AXI 事务变慢约 14.8%，不作为最终最快版本

## 3. FFN 32 路并行

- DDR burst：4 x 64 bit，32 字节对齐，不跨 4 KB 边界
- FFN 仿真周期：493,872 -> 327,864，减少 33.6%
- DSP：86 -> 102（46.36%）
- LUT：25,514（47.96%）
- BRAM：109（77.86%）
- 时序：WNS +0.332 ns，TNS 0，WHS +0.036 ns
- 6 层最后一行 hidden：96/96 个 32 位字一致
- 45 字符：17.202 s，逐字符一致，较 FFN16 提升 1.48 倍
- 200 字符：71.075 s，逐字符一致，较 FFN16 提升 1.39 倍

## 最终选择

最终板上版本采用单时钟 75 MHz、FFN 跨组预取、FFN 32 路并行。双时钟版本保留为实验备份，不替换最终版本。
